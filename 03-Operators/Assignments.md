# Assignment 3 — Even or Odd

### Problem

Store an integer in a variable.

Use the **Modulus (`%`)** operator to determine whether the number is **Even** or **Odd**.

### Example

```text
Number : 18

Remainder when divided by 2 : 0

The number is Even.
```

Try another number:

```text
Number : 13

Remainder when divided by 2 : 1

The number is Odd.
```

# 🚀 Bonus Challenge

Without using a calculator, predict the output of the following code before running it.


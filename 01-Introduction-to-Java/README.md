# Introduction to Java

This section introduces the basic structure of a Java program.

Before learning variables, loops, or Object-Oriented Programming, it is important to understand how a Java program is organized and how execution begins.

## Concepts Covered

* What is a Class?
* What is the `main()` Method?
* Understanding Comments
* Printing Output to the Console
* Curly Braces and Program Structure
* Semicolons and Statements

## Learning Objective

By the end of this section, you should be able to:

* Write your first Java program
* Understand the role of a class
* Understand the purpose of the `main()` method
* Display output using `System.out.println()`
* Recognize the basic structure of every Java application

## Example Program

The file `HelloWorld.java` demonstrates the simplest Java program.

Expected Output:

```text
Hello, World!
```

## Key Takeaway

Every Java application starts execution from the `main()` method, which acts as the entry point of the program.
